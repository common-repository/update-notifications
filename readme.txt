== Description ==

Check if your installation of wordpress are updates (core wordpress, themes and plugins). If so send an email.
You can also send a notification to other email addresses besides the administrator. 
You can also disable the receipt of notifications by type (eg core wordpress, plugins, themes).
You can set the length of time that will be in control.

== Installation ==

1. Upload the directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you want to change the setting of the plugin go to 'Update Notifications'.
